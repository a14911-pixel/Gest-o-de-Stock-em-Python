# 📦 Sistema de Gestão de Stock em Python

Um simples e funcional sistema de gestão de stock desenvolvido em Python, ideal para pequenos inventários ou para fins educativos. O programa permite adicionar materiais, consultar quantidades, atualizar entradas/saídas e visualizar o estado geral do stock.

---

## 🖥️ Funcionalidades

- ➕ **Adicionar Material**  
  Regista um novo item no stock com uma quantidade inicial.

- 🔍 **Consultar Stock**  
  Verifica a quantidade disponível de um material específico.

- ♻️ **Atualizar Stock**  
  Permite adicionar ou remover unidades de um material existente.

- 📋 **Exibir Stock Geral**  
  Mostra todos os materiais registados e respetivas quantidades.

- 🚪 **Sair do Programa**  
  Encerra a aplicação de forma segura.
